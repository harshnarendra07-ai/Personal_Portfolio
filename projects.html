AI Driven Portfolio

AI Driven Web Portfolio

Cinematic Experience & Architecture

Project Overview

A premium, cinematic personal portfolio architected with AI-assisted workflows. Emphasizes modern design principles, scroll-intercepting animations, and dynamic typewriter aesthetics.

Core Stack

• HTML5 / CSS3 / Vanilla JS
• Lenis (Smooth Scroll)
• Canvas API Particles
• Node.js & Express Backend

1. Inspiration & Concept

The vision was to create a luxury, "award-winning agency" feel. Moving away from generic templates, the site utilizes a stark dark theme, perfectly contrasted with pure white entrance typography and cinematic Canvas API particle layers.

2. Key Design Principles

Data-Driven Aesthetics: Clean lines and strict 8px grid systems map visually to analytical precision.
Accessibility: Semantic HTML5 and ARIA labels ensure the interactive dashboards and custom controls remain navigable for all users.

3. Custom Web Effects

Features a custom JavaScript scroll-interceptor that locks the viewport upon entry, displaying a sequence of CSS `@keyframes` manipulating `max-width` to create an authentic typewriter and blinking cursor effect before revealing the main layout.
